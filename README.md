🧠 Pedra, Papel ou Tesoura 🎮
Este é um simples jogo de "Pedra, Papel ou Tesoura" desenvolvido com HTML, CSS e JavaScript. O usuário pode jogar contra o computador, com placar de vitórias, derrotas e empates.
